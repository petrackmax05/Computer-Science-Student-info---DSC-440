# Academic Performance Analysis of Computer Science Students

## Description
This project analyzes academic performance data for computer science students using machine learning techniques to identify key factors that influence student success. The analysis includes exploratory data analysis (EDA), feature engineering, and predictive modeling to classify students as high-performing or low-performing based on GPA.

A Random Forest classifier and Logistic Regression model are implemented to evaluate predictive performance, while visualizations provide insight into trends and relationships within the data.

## Project Features

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
  - GPA distribution visualization
  - Correlation heatmap
- Feature engineering (binary classification of high-performing students)
- Machine learning models:
  - Random Forest Classifier
  - Logistic Regression
- Model evaluation:
  - Accuracy score
  - Classification report
  - Confusion matrix
  - ROC curve and AUC
  - Cross-validation
- Feature importance analysis

---

## Results

- Random Forest model demonstrated strong predictive performance
- Logistic Regression provided a comparable baseline model
- Feature importance analysis identified key academic and behavioral factors influencing student success
- Cross-validation confirmed model reliability and consistency
